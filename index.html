Built for selling

This site is designed for GitHub Pages: fast, modern, and easy to update. You can add products by editing a single list in app.js.

  • ✓ Product cards + modal
  • ✓ Cart drawer + totals
  • ✓ Search + filters
  • ✓ Mobile responsive

How you’ll take orders

Since GitHub Pages is static, this demo includes a “Checkout” button that can either (1) open a custom checkout link, or (2) generate a simple order summary you can copy into DMs. You can also replace it later with Shopify/Stripe links.

FAQ

Quick answers for running this on GitHub Pages.

How do I add products?
Edit the PRODUCTS array in app.js. Add name, price, images, category, and status.
Can I accept payments?
Yes—swap the demo checkout with a Stripe Payment Link, Shopify, Etsy, or a “DM to order” flow.
Will the cart save?
Yes. The cart uses localStorage, so it persists between refreshes on the same device.
